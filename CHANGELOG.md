# Changelog

All notable changes to this project are documented in this file.

The format follows Keep a Changelog and Semantic Versioning.

## [0.1.0] - 2026-06-28

### Added

- Repository scaffold and baseline toolchain.
- FastAPI app core with health/readiness endpoints.
- SQLAlchemy models and Alembic migrations.
- Authentication and role-based authorization modules.
- Backend domain routers, service, and repository layers.
- File storage API and dashboard aggregate endpoints.
- Frontend shell, routing, auth provider, protected routes, and API/query infrastructure.
- Frontend feature pages for home, dashboards, admissions, grades, assignments.
- Test suites for backend unit/integration and frontend unit/e2e.
- CI workflow and root Makefile quality gates.
- Dockerfiles, docker-compose stack, postgres init scripts, deployment runbook.
- Documentation package: API guide, user guide, architecture overview, release checklist, release policy.
